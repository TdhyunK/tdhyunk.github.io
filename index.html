Autocal

Tech Stack: ReactJS, React-Router, NodeJS, Webpack, NLP (In-Progress)

Autocal is a project that I began working on with two friends. It solves the problem of information overload by reading through emails, extracting event information (what, when, where) using NLP, and populating a personal calendar that contains all of the event information. You never have to miss an event again. As this is an ongoing project, I cannot post the code publicly, but feel free to email me if you would like to learn more about it.
