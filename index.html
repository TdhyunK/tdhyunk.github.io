K-Means Image Processing

This application takes an image and compresses it by grouping pixels together and using k-means to find an accurate color to represent the group of pixels.

Graphical Editor

Huffman Encoding

This program compresses the contents of a file through the use of Huffman Encoding. This file compression is accomplished by replacing frequently occuring characters with shorter codes. The characters were encoded with variable-lengths.

Kevin Bacon Game

Using data from a given file containing actors, movies, and actor-to-movie relationships, this program calculates how many degrees a given actor is separated from Kevin Bacon. The degree of separation is the actor's Kevin Bacon number. Breadth-first-search was used to find an actor's Kevin Bacon number.

Artificial Intelligence Connect 4

For this project, I created a game of connect 4 with an artificial-intelligence computer player with varying levels of difficulty. The computer player chooses its move based off of the results of alpha-beta pruning.

Tiny Search Engine

The tiny search engine is a search engine written in C. The tiny search engine has three components: the crawler, indexer, and querier. Show More.

The crawler is provided a seed URL and "crawls" through the seed URL to find any other web pages that are linked to it. The indexer uses the results of the crawler to generate an inverted index file that contains every word that was found in the web pages and the frequency of each word. Lastly, the querier handles search queries from the command line. Words that are searched together are handled with either a space or "AND" between the words, whereas words that must be searched separately are seprated with "OR" in between the words. Show less.

Mission Incomputable

Mission Incomputable is a game that I created with the use of a Pebble smartwatch written in C. The premise of the game is that there are code drops scattered at a certain location, and agents must capture the code drops while at the same time avoiding being neutralized by an opposing team's agent. Show More.

There are three components to the game: a guide agent, a field agent, and a gameserver. The guide agent is the individual that walks around campus with the Pebble smartwatch and captures code and neutralizes other players. The guide agent provides the field agent with hints regarding the location of the code drops and other information. The game server holds all of the information of the game, such as the location of the code drops, location of both guide agents and field agents, and which team has captured the most code drops.
